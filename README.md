This project helps validate whether your BIP39 Seed Phrase is correct and provides an additional way to ensure a secure backup by converting Stackbit and Tyneseed.

Features and Characteristics:
1 - Validates Tyneseed
2 - Validates Stackbit
3 - Supports only 12 or 24 words
4 - Validates the number of selected words
5 - Verifies whether the checksum is correct
6 - Stackbit disables fields to minimize incorrect selections
7 - Works offline with no external scripts
